## 二、为什么做 KPAX

### 市场洞察

1. 多 Agent 辩论赛道已有玩家（71vote、arbitrIQ、Verdikt、Suprmind），但没有赢家
2. 存在明确的空白地带：71vote 太轻（纯角色扮演），arbitrIQ 太重（49欧/次，面向高管）
3. 没有一个产品同时做到：有深度 + 普通人用得起 + 一句话就能开始
4. 预测市场工具层（Polyseer、PolyEdge）已有人做，我们不做预测市场本身

### 核心判断

- 技术没有护城河（LLM API + prompt，谁都能做）
- 学术数据没有护城河（OpenAlex 公开）
- 现阶段拼的是速度和执行力，不是护城河
- 护城河会在增长过程中形成（用户数据、Agent 历史表现、社区）

### 我们的位置

```
轻量无深度          有深度但昂贵
71vote -------- [KPAX] -------- arbitrIQ
(免费/角色扮演)    (中间地带)     (49欧/次/高管)
```

## 三、KPAX 是什么

### 一句话

**多个 AI 专家帮你辩论分析复杂问题，给你一份结构化的多角度分析报告。**

### 名字来源

致敬电影 K-PAX (2001)——一个声称来自外星的人，没人知道他说的是真是假，但他看问题的角度比所有人都深。KPAX 不给你答案，给你一个你没想过的角度。

### 用户场景

- "我该不该现在买房？"
- "这个创业方向靠谱吗？"
- "35岁该不该转行做AI？"
- "该不该投资新能源ETF？"
- "孩子该学钢琴还是编程？"
- 拿分析结果去 Polymarket 下注——那是你的事

### 和竞品的区别

| | KPAX | 71vote | arbitrIQ | 直接问 ChatGPT |
|---|------|--------|----------|---------------|
| 深度 | 有学术文献支撑 | 纯角色扮演 | 有 web search | 单一视角 |
| 价格 | $5-15/月 | 免费 | 49欧/次 | $20/月 |
| 门槛 | 一句话输入 | 一句话输入 | 上传文件 | 一句话输入 |
| 输出 | 结构化报告 | 讨论文字 | 决策报告 | 一段回答 |
| 多角度 | 3-5个专家辩论 | 20+角色讨论 | 2方对抗 | 1个视角 |

## 四、核心流程

```
用户一句话输入
    ↓
AI 自动识别相关领域 + 生成专家阵容
    ↓
3-5 个 AI 专家进行 2-3 轮结构化辩论
    ↓
输出：一份一页纸分析报告
  - 问题的多角度拆解
  - 主要支持论点（附依据）
  - 主要风险/反对论点（附依据）
  - 如果要深入，建议的 2-3 个方向
  - 相关学术研究引用（来自 OpenAlex）
```

**全程不超过 3 分钟。**

## 五、技术架构（复用 Agent X Lab）

从 Agent X Lab 1.0 复用：
- 多 Agent 辩论引擎（debate_engine.py）
- AI provider 层（LiteLLM 多模型支持）
- 知识图谱数据层（OpenAlex 4516 topics）
- Token 配额系统

需要新做：
- 通用专家角色生成（不只是学科学者，要能生成经济学家/律师/心理学家等）
- 问题解析器（一句话 → 识别涉及哪些领域 → 自动配置专家）
- 报告生成器（辩论结果 → 结构化一页纸报告）
- 全新前端（极简，一个输入框为核心）

## 六、Phase 1 计划（最小可用版本）

### 目标
让 10 个人用完之后，至少 3 个愿意用第二次。

### 范围
- 一个页面：输入框 + 报告展示
- 后端：问题解析 + 专家生成 + 2轮辩论 + 报告输出
- 不做：登录、支付、社区、历史记录
- 部署到一个可访问的地址

### 不做的事
- 不做预测市场集成
- 不做社区
- 不做支付
- 不做移动端
