# 工程项目

本目录存放从研究中衍生的工程实践项目。每个项目是独立文件夹，内部结构由项目本身决定。

## 项目索引

| 项目 | 状态 | 关联方向 | 说明 |
|------|------|----------|------|
| [knowledge-graph](knowledge-graph/) | 开发中 | 全方向交叉 | 交叉学科知识图谱可视化平台（React + FastAPI + PostgreSQL + D3.js） |

## 新建项目规范

1. 在本目录下创建以项目名命名的文件夹
2. 项目根目录必须包含 `README.md`，声明：
   - 项目目标
   - 技术栈
   - 与 `research/` 中哪些方向相关
3. 在上方索引表中添加一行
